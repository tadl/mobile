==============================================
WINK DOCUMENTATION
==============================================

GOALS
==============================================
- Automatically generate the documentation from the source code

HOW TO USE
==============================================
- Wink documentation is generated using JsDoc. For more info, visit te JSDoc project homepage at http://code.google.com/p/jsdoc-toolkit/

- To build the documentation, from the "/utils/doc" folder, run the following command line:

		java -jar libs/jsdoc-toolkit-2.4.0/jsrun.jar libs/jsdoc-toolkit-2.4.0/app/run.js -c=conf/wink.conf
